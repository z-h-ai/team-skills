# 视频拆解直接复刻流程说明

## 问题背景

用户有时不走改写流程（script-rewriter），而是：
1. 使用 `video-analyzer` 拆解视频
2. 直接用 `ai-video-generator` 复刻视频

**之前的问题**：拆解时提取了台词，但复刻时生成的视频没有包含台词。

## 解决方案

在 `video-analyzer` 的输出中新增 `scenes` 字段，将台词整合到 `prompt` 中，使输出可以直接对接 `ai-video-generator`。

## 修改内容

### 1. SKILL.md 文档修改

- 在"输出格式"中增加了 `scenes` 字段说明
- 新增"生成 scenes 字段"章节，说明转换规则
- 新增"使用场景"章节，区分改写流程和直接复刻流程
- 增加完整的输出示例

### 2. analyze_video.py 脚本修改

新增三个函数：

- `generate_scenes_from_structure()`: 将 structure 转换为 scenes 格式
- `calculate_duration()`: 从时间范围（如 "0-3秒"）计算 duration
- `build_prompt()`: 构建 prompt，格式为"人物描述+说：\"台词\"+景别"

在分析完成后自动调用这些函数生成 `scenes` 字段。

## 使用方式

### 直接复刻流程（不改写）

```bash
# 步骤1: 拆解视频（必须使用 --transcribe 参数提取台词）
python scripts/analyze_video.py video.mp4 --transcribe --output result.json

# 步骤2: 直接生成视频（无需改写）
python scripts/generate_all.py --script result.json --output-dir ./clips

# 步骤3: 拼接视频
./scripts/concat_videos.sh ./clips output.mp4
```

### 改写流程（原流程不变）

```bash
# 步骤1: 拆解视频生成模板
python scripts/analyze_video.py video.mp4 --transcribe --output template.json

# 步骤2: 使用 script-rewriter 改写
# （通过 AI 交互改写，输出 new_script.json）

# 步骤3: 生成视频
python scripts/generate_all.py --script new_script.json --output-dir ./clips

# 步骤4: 拼接视频
./scripts/concat_videos.sh ./clips output.mp4
```

## 输出格式对比

### video-analyzer 输出（新增 scenes 字段）

```json
{
  "duration": 60,
  "ratio": "9:16",
  "structure": [...],
  "reusable_template": {...},
  "scenes": [
    {"prompt": "职场女性对着镜头说：\"这3个技巧，90%的人都不知道\"，特写镜头", "duration": 3},
    {"prompt": "职场女性继续说：\"你每天都在加班？其实效率比时间更重要\"，中景", "duration": 5},
    ...
  ],
  "resolution": "720p",
  "generate_audio": true,
  "transcription": {...}
}
```

### ai-video-generator 输入（与 video-analyzer 输出兼容）

```json
{
  "scenes": [
    {"prompt": "...", "duration": 3}
  ],
  "resolution": "720p",
  "ratio": "9:16",
  "generate_audio": true
}
```

## 关键改进

1. **台词保留**：structure 中的 `dialogue` 字段会被整合到 scenes 的 `prompt` 中
2. **格式统一**：台词用双引号包裹，格式为 `说："台词内容"`
3. **直接对接**：video-analyzer 的输出可以直接作为 ai-video-generator 的输入
4. **向后兼容**：原有的 structure 和 reusable_template 字段保留，不影响改写流程

## 注意事项

1. **必须使用 --transcribe 参数**：直接复刻时必须提取台词
2. **台词准确性**：依赖语音识别 API 的准确性
3. **prompt 质量**：自动生成的 prompt 可能需要人工微调
4. **音画同步**：Seedance 1.5 pro 会根据 prompt 中的台词自动生成同步音频

### 生成效果

- 7个视频片段，每个片段包含完整台词

## 后续优化方向

1. **台词时间戳匹配**：使用 transcription 的精确时间戳匹配台词
2. **prompt 优化**：根据视频类型自动优化 prompt 格式
3. **情感标注**：自动识别台词的情感（认真/微笑/严肃）并添加到 prompt
4. **动作标注**：自动识别关键动作（挥手/点头/手势）并添加到 prompt
