# 短视频脚本改写

基于爆款脚本模板，改写生成新主题脚本。

## 核心原则

- 结构通用（黄金3秒、冲突设置等）
- 内容独特（主题、台词、案例）
- 保持节奏和逻辑，替换具体内容

## 工作流程

1. 解析脚本模板（来自video-analyzer）
2. **确定视频比例**：
   - 如果有参考视频分析结果，自动使用参考视频的 `ratio`
   - 如果没有参考视频，使用 AskUserQuestion 询问用户选择视频比例
3. **检测产品图片**（可选）：
   - 如果用户上传了产品图片，记录图片路径
   - 自动调整第一个场景的prompt，适配产品图片展示
4. 确定新主题、受众、语气
5. 逐部分内容替换
6. 输出新脚本（供ai-video-generator使用）
   - 如果有产品图片，在JSON中添加 `first_frame` 字段

## 改写规则

### 黄金3秒
```
原："这3个[AI工具]，90%的人都不知道"
新："这3个[时间管理技巧]，90%的人都不知道"

保持：数字开头、悬念感、痛点
替换：具体主题词
```

### 冲突设置
```
原："你还在用[ChatGPT]？其实[这些工具]更强大"
新："你每天都在[加班]？其实[效率]比[时间]更重要"

保持：反问句式、对比结构
替换：具体场景、对比对象
```

### 内容展开
```
保持：3个要点结构、每个要点时长
替换：具体内容、案例、演示
```

### 反转高潮
```
原："其实最重要的不是[工具]，而是[如何使用]"
新："其实最重要的不是[管理时间]，而是[管理精力]"

保持：转折句式、深层洞察
替换：表面vs深层对比
```

### CTA引导
```
原："关注我，每天分享一个[AI技巧]"
新："关注我，每天分享一个[职场技巧]"

保持：行动号召、承诺句式
替换：主题领域
```

## 输出格式

生成可直接用于ai-video-generator的JSON：

```json
{
  "first_frame": "path/to/product.jpg",  // 可选：产品图片路径（用户上传时添加）
  "scenes": [
    {"prompt": "人物描述+台词+表情+景别", "duration": 4},
    {"prompt": "...", "duration": 4},
    {"prompt": "...", "duration": 12},
    {"prompt": "...", "duration": 12},
    {"prompt": "...", "duration": 12},
    {"prompt": "...", "duration": 6},
    {"prompt": "...", "duration": 6}
  ],
  "resolution": "720p",
  "ratio": "16:9",
  "generate_audio": true
}
```

**产品图片说明**：
- `first_frame` 字段是**可选的**，仅在用户上传产品图片时添加
- 如果提供了产品图片，第一个场景会使用图生视频模式（基于产品图片生成）
- 产品图片路径可以是本地路径或URL

**视频比例说明**:
- 如果有参考视频分析结果，**自动使用参考视频的 `ratio`**
- 如果没有参考视频，**使用 AskUserQuestion 询问用户选择视频比例**
- 可选比例：
  - `"16:9"` - 横屏（适合YouTube、B站）
  - `"9:16"` - 竖屏（适合抖音、快手、视频号）
  - `"1:1"` - 方形（适合Instagram、小红书）

**询问示例**（当没有参考视频时）:
```
使用 AskUserQuestion 工具询问：
- question: "请选择视频比例"
- header: "视频比例"
- options:
  - label: "横屏 16:9", description: "适合YouTube、B站等横屏平台"
  - label: "竖屏 9:16", description: "适合抖音、快手、视频号等竖屏平台"
  - label: "方形 1:1", description: "适合Instagram、小红书等方形平台"
```

## 产品图片融合

### 使用场景

当用户需要在视频中展示产品时，可以上传产品图片：
- 产品带货视频（如卖面罩、卖暖手宝）
- 产品评测视频
- 产品教程视频

### 工作流程

1. **用户上传产品图片**（可选）
   - 支持JPG、PNG格式
   - 建议尺寸：根据视频比例调整
     - 16:9 → 1920x1080
     - 9:16 → 1080x1920
     - 1:1 → 1080x1080

2. **自动调整第一个场景**
   - 原prompt：`"职场女性对着镜头说：'这3个技巧...'，特写"`
   - 有产品图片时：`"产品特写展示，清晰的产品细节，[产品名称]"`

3. **输出JSON包含产品图片**
   ```json
   {
     "first_frame": "path/to/product.jpg",
     "scenes": [
       {"prompt": "产品特写展示，清晰的产品细节", "duration": 5},
       ...
     ]
   }
   ```

### 示例

**场景1：卖面罩视频**
```
用户上传：暖手宝产品图片
原视频：卖面罩（竖屏9:16）
新主题：卖暖手宝

输出脚本：
{
  "first_frame": "warmhand.jpg",
  "scenes": [
    {"prompt": "暖手宝产品特写，清晰展示外观和细节", "duration": 5},
    {"prompt": "手持暖手宝，按下开关按钮特写", "duration": 4},
    {"prompt": "暖手宝发光发热，温暖效果展示", "duration": 6},
    ...
  ],
  "ratio": "9:16"
}
```

**场景2：无产品图片**
```
用户未上传产品图片
原视频：AI工具推荐（横屏16:9）
新主题：时间管理技巧

输出脚本：
{
  "scenes": [
    {"prompt": "职场女性对着镜头说：'这3个技巧...'，特写", "duration": 4},
    ...
  ],
  "ratio": "16:9"
}
```
